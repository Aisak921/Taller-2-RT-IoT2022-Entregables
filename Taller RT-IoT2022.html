Dataset y Estructura de Archivos

Se trabaja con tres archivos del repositorio Taller-2-RT-IoT2022:

  • train_data.csv: Datos de entrenamiento con etiquetas (Attack_type)
  • test_data.csv: Datos de prueba sin etiquetas
  • test_label_data.csv: Etiquetas de validacin (opcional para verificacin post-competencia)

El dataset contiene trfico capturado en un testbed controlado usando Wireshark, con caractersticas extradas por CICFlowmeter (puertos, protocolo, IAT, flags TCP, etc.).
